A Vue.js-based coding practice platform that simulates real technical interviews with AI-powered evaluation.

### Key Features
- AI interviewer with real-time feedback and scoring
- In-browser code sandbox with multi-language support
- Problem categorization by difficulty and topic
- Performance tracking and progress analytics
