A .NET WebAPI template focused on authentication and authorization, powered by the CoreFX SDK.

### Key Features
- JWT authentication with refresh token rotation
- Dapper ORM for performant data access
- Redis caching layer
- Role-based authorization
- Swagger/OpenAPI documentation
