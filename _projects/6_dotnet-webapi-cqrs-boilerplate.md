A .NET WebAPI template implementing the CQRS pattern with MediatR, built on the CoreFX SDK.

### Key Features
- MediatR for Command/Query Responsibility Segregation
- FluentValidation for request validation pipeline
- JWT authentication
- Email notifications via MailKit
- Docker multi-stage build
- xUnit test coverage
- Multi-targets `net8.0` + `net10.0`
