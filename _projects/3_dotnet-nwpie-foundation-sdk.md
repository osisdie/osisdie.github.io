A comprehensive, modular .NET SDK providing reusable building blocks for enterprise applications.

### Components
- **Caching** -- Redis, MemoryCache with circuit breaker
- **Auth** -- JWT, API key, role-based access
- **Messaging** -- SQS, SNS, event-driven patterns
- **Storage** -- S3, Azure Blob, local file system
- **DataAccess** -- Dapper, Entity Framework abstractions
- **Logging** -- Structured logging with Serilog
- And 10+ more modules

### Targets
- `netstandard2.1` for broad compatibility
- `net8.0` / `net10.0` for latest features
