A TypeScript-based bridge that connects Claude Code CLI to popular messaging platforms, enabling bidirectional remote control of your AI agent from anywhere.

### Key Features
- Multi-platform support: Telegram, Discord, Slack, LINE, WhatsApp
- Bidirectional messaging with full Claude Code capabilities
- Session management and authentication
- Easy deployment with Docker or Node.js
