A production-ready .NET WebAPI template built on the CoreFX SDK, designed for cloud-native deployment.

### Key Features
- Structured API responses with consistent error handling
- Request/response logging middleware
- Redis caching with circuit-breaker pattern
- Docker multi-stage build for optimized images
- Kubernetes and Minikube deployment manifests
- Multi-targets `net8.0` + `net10.0`
